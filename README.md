**Project Description:**  
DrawAgain is a calm, supportive web application designed to help artists overcome creative block. It helps users build a consistent drawing habit through low-pressure goals, guided focus sessions, and inspiring prompts, all wrapped in a gentle, non-judgmental interface.

### The Problem & Solution
**The Problem:** many aspiring artists struggle with "blank page syndrome," perfectionism, and the pressure to produce "good" art, which often leads to inconsistent practice and eventual burnout.

**The Solution:** DrawAgain removes the pressure by focusing on the *act* of starting rather than the result. By offering small, manageable weekly goals, a distraction-free timer with calming modes, and simple prompts, it creates a safe digital space where creativity can flow without anxiety.

## Technical Details
**Technologies Used:**

- **Routing:** React Router DOM
- **State Management:** React Context API + Local Storage

---

## Features
- **Gentle Goal Setting**: A flexible weekly goal system that supports custom targets and encourages small starts (e.g., "Just begin gently").
- **Immersive Focus Timer**: A distraction-free drawing timer with distinct modes (**Focus**, **Flow**, **Calm**) featuring ambient animations and encouraging messages.
- **Creative Prompt Generator**: A tool that provides diverse drawing ideas and "creative sparks" to instantly cure creative block.
- **Smart Progress Tracking**: A visual weekly planner that tracks completed drawings and automatically resets each week to keep motivation fresh.
